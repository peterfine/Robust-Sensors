Detailed Description

This class represents a homeostatic, Braitenburg style controller, as defined in the second experiment of: Di Paolo, E. A., (2003). Organismically-inspired robotics: Homeostatic adaptation and natural teleology beyond the closed sensorimotor loop, in: K. Murase & T. Asakura (Eds) Dynamical Systems Approach to Embodiment and Sociality, Advanced Knowledge International, Adelaide, Australia, pp 19 - 42.

Note it is not defined by a Genotype, and will quit the program if reconstruction is attempted!
